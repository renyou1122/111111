1. 什麼是 Git？ (核心工具)

  • 這是什麼？ Git 是一個分散式版本控制系統 (Version Control System)。

    簡單比喻： 想像你在玩遊戲時的「存檔 (Save)」功能。Git 讓你可以在寫程式的過程中隨時「存檔」，如果寫壞了，可以隨時「讀檔」回到之前的狀態，而不會遺失檔案。

  • 為什麼要使用？

    • 時光機功能： 寫程式寫錯了導致整個專案壞掉？Git 可以讓你一鍵回到昨天正常的版本。

    • 多人合作： 多人同時修改同一個檔案時，Git 可以協助合併大家的修改，避免互相覆蓋。

    • 離線工作： 不像雲端硬碟需要網路，Git 的版本控制在你的電腦（本機）上就能運作。

  • 如何使用？ 通常透過 Cmd (命令列) 輸入指令。

    • git init：在資料夾內開始錄製版本。

    • git add：選擇要存檔的檔案。

    • git commit：確認存檔。
