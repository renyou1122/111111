(一) 前端、後端、 Flask料庫:

                         eternal_realm_game/

                         models.py               ← Peewee資料庫模型

                         requirements.txt     ← Python依賴套件 

                         config.py                ← 應用程式設定

                         static/                    ← 靜態檔案

                         js/

                         (main.js )              ← 遊戲主邏輯 

                         ( survival-game.js)← 生存遊戲邏輯 

                         (brython/ )            ← Brython庫

                         images/                 ← 圖片資源 

                         templates/            ← Flask模板 

                         index.html            ← 遊戲主頁面 (  你的游戏主页面，需要登录访问)

                         uploads/              ← 上傳檔案 

                         ( backgrounds/ )  ← 使用者上傳背景圖片

(二) CMD  啟動步驟:

1.設置位置: 

cd “C:\Users\user\eternal_realm_game>”

or

cd eternal_realm_game

之後會顯示【 C:\Users\user\eternal_realm_game> 】

2.確保已安裝Python 3.8+ : python –version

3.建立虛擬環境 python -m venv venv

4.啟動虛擬環境 Windows: venv\Scripts\activate 之後會顯示

【 (venv) C:\Users\user\eternal_realm_game> 】

5.安裝依賴套件: pip install -r requirements.txt

6.初始化資料庫 python models.py

7. 啟動開發伺服器 python app.py

(三) Ngrok 啟動步驟:

1.啟動桌面 啟動步驟: ngrok.exe : 點兩下打開

2.運行 ngrok 命令： ngrok http 6000

3. 在瀏覽器輸入 【 https://marry-slithery-simon.ngrok-free.dev 】
